Welcome to **Enat**, an interactive platform designed to help users learn and practice the Amharic language. Built with **React** and **Material UI**, Enat provides an engaging and intuitive experience for mastering Amharic words, phrases, and grammar through lessons, quizzes, and interactive exercises.

Enat includes a structured learning path, personalized progress tracking, and a leaderboard system to keep learners motivated. The platform also features JWT-based authentication for secure access and role-based permissions.
